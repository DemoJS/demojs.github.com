DemoJS demoparty - Paris - 1st & 2nd of July 2011

DemoJS party is a french demoparty focused on web technologies (JavaScript, Canvas, SVG, Audio... well every openweb-related piece of technology). The event is part of the Mozilla Labs demoparty initiative.

Demoscene? Demoparty?

The demoscene is a computer art subculture that specializes in producing demos, which are audio-visual presentations that run in real-time on a computer. Demosceners are coders, graphic artists, computer musicians... (more info)

A demoparty is an event that gathers demosceners and other computer enthusiasts to compete (more info).

DemoJS demoparty is a demoparty specialized in web technologies. We welcome both experienced demosceners and people who have never been to a demoparty.

If you are a beginner in web technologies, here are some links to help you. Also, you will find help at the party place, don't hesitate to ask people around you!

Compos

Open Web technologies only, no plug-ins, no extras. The demo should not have been released previously, should work on the last version of Firefox & Chrome, and should work offline.

  • Size limited - JavaScript 1k: play with a few bytes (The original); the shim file is here (exhaustive rules are in the source, view the source here)
    prod examples: "particle carriage" by @p01, more examples on js1k.com
  • Freestyle - One effect or more: no size-limit, just show us cool stuff using web technologies ;) (JS, webGL, canvas, ... and why not CSS3, GIF, PNG, SVG, or even audio)
    prod examples: webgl full demo, pure javascript+fun, canvas, css

Conferences and workshops

14h00 - 2D Animation Engines and Perf Experiments by Louis-Rémi Babé

The Open Web offers different options to create 2D effects and games: DOM manipulation, HTML generation, canvas 2D drawing and even WebGL.

In this session, we'll see how to create a 30loc Animation Engine able to use those different "backends", how they compare in performance and what are their other advantages/inconvenients.

about the speaker: Louis-Rémi is a long time Open Web enthusiast and jQuery contributor with a passion for performances and 2D graphics. He is constantly experimenting with new ways to improve animation performances and is now an intern at Mozilla Paris, writing technical blog posts and demos for hacks.mozilla.org.


15h00 - WebGL osgjs framework by Cédric Pinson

about the speaker: Cédric has been working in 3D software for over 12 years. He started off his career in the video game industry at Nemosoft in 2000 and then moved to Mekensleep in 2004. In 2007, he joined Outflop where he served as the project leader for 3D client technology. He is a contributor to the OpenSceneGraph project and the author and maintainer of osgAnimation, the OpenSceneGraph's animation nodekit. He contributed to the 2011 publication of OpenSceneGraph Beginner's Guide as a technical reviewer. Recently, with the introduction of WebGL, he created javascript version of OpenSceneGraph known as osgJS to bring this api to the web. He now does freelance work using OpenGL and WebGL technologies.


16h00 - Demoscene: from cracktros to browser demos by Jean-François Richard and Jean-Christophe Gatuingt

A brief history of the demoscene with an emphasis on the recent browser demos.

about the speakers: Jeffrey is an old-school demoscener, co-founder of the AmigaVibes portal and member of the Vitalmotion demogroup. Jean-Christophe is involved in the demoscene since 2009 with Adinpsz, demoscene.fr and now DemoJS.
