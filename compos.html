General rules

  • Open web technologies only (webGL, canvas, ...). No plug-ins, no extras.
  • The production should not have been publicly released previously
  • The production have to work on at least one major webbrowser (IE, Firefox, Chrome). Please tell us which browser to use to run the demo/intro. You are allowed to experiment on the nightly build of Firefox or Chrome, but please keep us informed.
  • You should only submit your own work. In particular, it is forbidden to submit copyrighted material.
  • The organizing team reserve the right to disqualify entries that are inappropriate (pornography, illegal content, ...).
  • The entries will be run on a high-end computer.
  • You can't start working on your production NOW. The production does NOT need to be made 100% at the party place (but it should not have been released previously)
  • Deadline is June, 30th, 20:00 CET
  • You can submit your entry at the party place or by email for remote entries: remote@demojs.org

Compos

Browser 1k

Play with a few bytes!

  • Maximum filesize is 1KB including the HTML container and all resources required to run the intro.
  • The production have to work offline. You are NOT allowed to retrieve content from the Internet
  • You should not use any operating system provided media files such as the "Media" and "Music Samples" directory
  • This year, you DO NOT have a free HTML shim.
  • Prod examples: Glenz1k (DemoJS 2011 winner), Creations (DemoJS 2011), gl1k cotton candy (DemoJS 2011)

Browser 10k

Try to put cool visuals and music in 10k!

  • Maximum filesize is 10KB including the HTML container and all resources required to run the intro.
  • The production have to work offline. You are NOT allowed to retrieve content from the Internet
  • You should not use any operating system provided media files such as the "Media" and "Music Samples" directory
  • Prod examples: Fabrik (DemoJS 2012 invitation - only 4KB)

Wild

Cool hacks that are not demos: games, datavisualisation, ...

  • No maximum filesize. However dont try to run a movie in an iframe.
  • You CAN retrieve external content (from the Internet). Actually, hacking with APIs is encouraged.
  • Prod examples: The 1d1k game (DemoJS 2011), Datagalaxy (DemoJS 2011), Particles.js (DemoJS 2011)

Remote entries

You can enter the competitions even if you won't be present at the party.

Send us your(s) production(s) by email: remote@demojs.org

Please include in your email:

  • the production title
  • author/group name
  • (optionnal) a message about your production, that will be read to the live audience (of course, you can do yourself an audio or video if you prefer)
